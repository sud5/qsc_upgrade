The Pass Grade quiz access rule

This quiz access rule prevents students who have achieved a passing grade from attempting the quiz again.

It can be used with versions 2.2 of Moodle, or later.

To install this plugin checkout the files into mod/quiz/accessrule/passgrade:

e.g. git clone git@github.com:catalyst/moodle-quizaccess_passgrade.git mod/quiz/accessrule/passgrade
